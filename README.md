Интерактивное VR-приложение для наглядного погружения в геометрию и
топологию 3-мерных многообразий (3-manifolds).
Пользователь должен свободно перемещаться внутри различных пространств,
наблюдать эффекты идентификаций (gluing), кривизны и топологических особенностей, 
что делает абстрактные концепции интуитивно понятными.
